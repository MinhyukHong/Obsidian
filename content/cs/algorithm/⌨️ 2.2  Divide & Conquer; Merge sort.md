
## 합병 정렬
---
- 문제: $n$개의 정수를 비내림차순으로 정렬
- 입력: 정수 $n$, 크기가 $n$인 배열 $S[1..n]$
- 출력: 비내림차순으로 정렬된 배열 $S[1..n]$

> *Pseudo Code*
```
void mergesort(int n, keytype S[]) {
	const int h = n/2, m = n-h;
	keytype U[1..h], V[1..m];
	
	if(n > 1) {
		copy S[1] through S[h] to U[1] through U[h];
		copy S[h+1] through S[n] to V[1] through V[m];
		mergesort(h, U);
		mergesort(m, V);
		merge(h, m, U, V, S);
	}
} 
```

```
void merge(int h, int m, const keytype U[], const keytype V[], keytype S[]) {
	index i, j, k;
	i = 1; j = 1; k = 1;
	while(i<=h && j<=m) {
		if(U[i] < V[j]) {
			S[k] = U[i];
			i++;
		}
		else {
			S[k] = V[j];
			j++;
		}
		k++;
	}
	if(i > h)
		copy V[j] through V[m] to S[k] through S[h+m];
	else
		copy U[i] through U[h] to S[k] through S[h+m];
}
```


## `merge()` 알고리즘 시간복잡도 분석 - 최악의 경우
---
- 단위 연산: $U[i]$와 $V[j]$의 비교
- 입력 크기: 2개의 입력 배열에 각각 들어 있는 항목의 개수: $h$와 $m$
- 분석: $i=h+1$이고, $j=m$인 상태로 loop에서 빠져 나가는 때가 최악의 경우로서(==V에 처음 m-1개의 항목이 S의 앞부분에 위치하고, U에 있는 h개의 항목이 그 뒤에 위치하는 경우==), 이 때 단위연산의 실행 횟수는 $h+m-1$이다. 따라서, 최악의 경우 합병하는 시간복잡도는 $W(h,m)=h+m-1$.
- example.
	- U: 4, 5, 6, 7
	- V: 1, 2, 3, 8


## `mergesort()` 알고리즘 시간복잡도 분석 - 최악의 경우
---
- 단위 연산: 합병 알고리즘 merge에서 발생하는 비교
- 입력 크기: 배열 $S$에 들어있는 항목의 개수 $n$
- 분석: 최악의 경우 수행시간은 $W(h,m)=h+m-1$이 된다. 여기서 $W(h)$는 $U$를, $W(m)$은 $V$를 정렬하는데 걸리는 시간이다. 그리고 $h+m-1$은 합병하는데 걸리는 시간이다. 정수 $n$을 $2^k(k\ge 1)$이라고 가정하면, $h=\frac {n}{2}$, $m=\frac {n}{2}$이 된다.  따라서 최악의 경우 재현식은:
	- $W(n)=2W(\frac {n}{2})+n-1$, $n>1$이고, $n=2^k (k\ge 1)$
	- W(1) = 0
	- 재현식의 해는 도사정리를 이용하여 $W(n)\in \Theta(nlgn)$


## 공간복잡도 분석
---
- 추가적인 저장장소를 사용하지 않고 정렬하는 알고리즘: **제자리정렬(in-place sort) 알고리즘**
- 합병정렬 알고리즘으로 제자리정렬 알고리즘이 아니다. 입력 배열 S 이외에 U와 V를 추가로 만들어서 사용한다.
- 재귀 호출이 종료될 때까지 상위 재귀 호출이 생성하는 공간이 유지되어야 한다.
- 추가적인 저장장소: `mergesort`를 재귀 호출할 때마다 크기가 S의 반이 되는 U와 V가 추가적으로 필요하다. (==`merge`는 추가 저장장소 불필요==).
- 처음 S의 크기가 n이면, 추가적으로 필요한 U와 V의 저장장소 크기의 합은 n이 된다. 다음 재귀 호출에는 $n/2$의 추가적으로 필요한 총 저장공간의 크기는
	- $n+\frac{n}{2}+\frac{n}{4}...=2n$, $2n \in \Theta(n)$


### 공간복잡도가 향상된 알고리즘
---
> *Pseudo Code*
```
void mergesort2(index low, index high) {
	index mid;
	if(low < high) {
		mid = (low+high) / 2;
		mergesort2(low, mid);
		mergesort2(mid+1, high);
		merge2(low, mid, high);
	}
}
...
mergesort2(1, n);
...
```

```
void merge2(index low, index mid, index high) {
	index i, j, k; keytype U[low..high]; // 합병 시 필요한 지역 배열
	i = low,; j = mid + 1; k = low;
	while(i<=mid && j<=high) {
		if(S[i] < S[j]) {
			U[k] = S[i];
			i++;
		}
		else {
			U[k] = S[j];
			j++;
		}
		k++;
	}
	if(i > mid)
		copy S[j] through S[high] to U[k] through U[high];
	else
		copy S[i] through S[mid] to U[k] through U[high];
	copy U[low] through U[high] to S[low] through S[high];
}
```
