
## 부분집합의 합 구하기
---
- n개의 item을 이용하여 item들의 무게의 합이 W가 되는 부분집합을 구한다.
	$$
	\Sigma_{i=1}^{n}w_ix_i=W
	$$
	$x_i=0$ or 1, for $i=1, n$

- For $S=\{1,4,6,8\}$, select items so that sum of the subset is 5.
- $S=\{a_1, a_2, ..., a_n\}$ 인 경우 생성가능한 부분집합
	- S의 멱집합, power set = $2^S$ 
	- $2^S=\{\{\}, \{a_1\}, \{a_2\}, \{a_1, a_2\}, ..., \{a_1, a_2, ..., a_n\}\}$

![Figure 1](Backtracking_ss_1.png)

- 무게가 증가하는 순으로 데이터를 정렬 -> 유망하지 않은지를 쉽게 판단할 수 있다.
- $w_{i+1}$ 는 i수준에서 남아있는 가장 가벼운 아이템의 무게.
- $w_{i+1}$ 를 넣을 수 없으면 i+1 이후는 고려할 필요가 없다.
- weight: 수준 i의 마디까지 포함된 무게의 합
- total: 남아 있는 아이템의 무게의 총 합
- $weight+w_{i+1}$ > W(if weight != W) or weight + total < W 이면 유망하지 않다.

>*Pseudo Code*
```
void sum_of_subsets(index i, int weight, int total){
	if(promising(i))
		if(weight == W)
			cout << include[1] through include[i];
		else{
			include[i+1]="yes";
			sum_of_subsets(i+1, weight+w[i+1], total-w[i+1]);
			include[i+1]="no";
			sum_of_subsets(i+1, weight, total-w[i+1]);
		}
}

bool promising(index i){
	return (weight+total>=W) && (weight == W || weight+w[i+1]<=W);
}
```
