
## 행렬곱셈 알고리즘
---
문제: 두 개의 $n*n$ 행렬의 곱 
입력: 양의 정수 $n$, 수의 2차원 배열 $A$와 $B$. 여기서 이 행렬의 행과 열은 모두 1부터 $n$까지의 첨자를 갖는다.
출력: $A$와 $B$의 곱이 되는 수의 2차원 배열 $C$. 이 행렬의 행과 열은 모두 1부터 $n$까지의 첨자를 갖는다.

> *Pseudo Code*
```
void matrixmult(int n, const number A[][], const number B[][], number C[][]) {
	index i, j, k;
	
	for(i=1; i<=n; i++)
		for(j=1; j<=n; j++) {
			C[i][j] = 0;
		for(k=1; k<=n; k++)
			C[i][j] = C[i][j] + A[i][k] * B[k][j];
		}
}
```


## 시간복잡도 분석
---
- 단위 연산: ==가장 안쪽 for 루프에 있는 곱셈==
- 입력 크기: 행과 열의 개수 $n$(*square matrix*)
- **모든 경우** 분석:
	- `for i` 루프는 항상 $n$번 수행. `for i` 루프 한 번 수행될 때마다 `for j` 루프는 항상 $n$번 수행
	- `for j` 루프가 한 번 수행될 때마다 `for k` 루프는 항상 $n$번 수행
	- $T(n)=n*n*n=n^3$
