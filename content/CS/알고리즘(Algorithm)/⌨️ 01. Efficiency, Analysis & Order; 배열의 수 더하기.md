
## 배열의 수 더하기
---
문제: $n$개의 수로 된 배열 $S$에 있는 모든 수의 합
입력: 양의 정수 $n$, 수의 배열 $S$(첨자는 1~$n$)
출력: $S$에 있는 수의 합, $sum$

> *Pseudo Code*
```
number sum(int n, const number S[]) {
	index i;
	number result;
	
	result = 0;
	for(i=1; i<=n; i++)
		result = result+S[i];
	return result;
}
```


## 시간복잡도 분석
---
- 단위 연산: ==덧셈==
- 입력 크기: 배열의 크기 $n$
- **모든 경우** 분석:
	- 배열 내용에 상관 없이 for 루프가 $n$번 반복된다.
	- 각 루프마다 덧셈이 1회 수행된다.
	- 따라서, $n$에 대해서 덧셈이 수행되는 총 횟수는 $T(n)=n$이다.

- 단위 연산: ==지정문==(for 루프의 첨자 지정문 포함)
- 입력 크기: 배열의 크기 $n$
- **모든 경우** 분석:
	- 배열 내용에 상관 없이 for 루프가 $n$번 반복된다.
	- 따라서, 지정문이 $T(n)=n+n+1$번 수행된다.
