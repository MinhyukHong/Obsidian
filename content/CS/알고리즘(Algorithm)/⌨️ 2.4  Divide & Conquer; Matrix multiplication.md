
## 단순한 행렬곱셈 알고리즘
---
- 문제: $n*n$ 크기의 행렬의 곱을 구하시오.
- 입력: 양수 $n, n*n$ 크기의 행렬 A와 B
- 출력: 행렬 A와 B의 곱인 C

> *Pseudo Code*
```
void matrixmult(int n, const number A[][], const number B[][], number C[][]) {
	index i, j, k;
	for(i=1; i<=n; i++) {
		C[i][j] = 0;
		for(k=1; k<=n; k++)
			C[i][j] = C[i][j] + A[i][k] * B[k][j];
	}
}
```


## 시간복잡도
---

### 분석 1
---
- 단위 연산: 가장 안쪽 루프에 있는 곱셈하는 연산
- 입력 크기: 행과 열의 수, $n$
- *모든 경우* 시간복잡도 분석: 총 곱셈의 횟수
	- $T(n)=n*n*n=n^3\in \Theta(n^3)$


### 분석 2 (알고리즘 일부 수정)
---
- 단위 연산: 가장 안쪽 루프에 있는 곱셈하는 연산
- 입력 크기: 행과 열의 수, $n$
- *모든 경우* 시간복잡도 분석: 총 곱셈의 횟수
	- $T(n)=(n-1)*n*n=n^3-n^2\in \Theta(n^3)$
	- $A[i][k]*B[k][j]$를 계산한 최초 값을 $C[i][j]$ 대신 사용한다면 계산이 $n-1$로 줄어든다.
	- 하지만 시간복잡도는 위와 마찬가지로 $\Theta(n^3)$.

## Strassen 알고리즘
---
- 문제: $n$이 2의 거듭제곱일 때, n x n크기의 두 행렬의 곱
- 입력: 정수 n, n x n 크기의 행렬 A와 B
- 출력: A와 B의 곱인 C

> *Pseudo Code*
```
void strassen(int n, n*n_matrix A, n*n_matrix B, n*n_matrix& C) {
	if(n <= threshold)
		단순한 알고리즘을 사용하여 계산;
	else {
		A를 4개의 부분 행렬로 분할;
		B를 4개의 부분 행렬로 분할;
		쉬트라쎈 방법을 사용하여 C = A * B를 계산;
		// 되부르는 호출의 예: strassen(n/2, A_11+A_22, B_11+B_22, M_1)
	}
}
```

- ==threshold==: 두 알고리즘의 효율성이 교차하는 문제의 크기


## 시간복잡도
---

### 1. 단순한 방법
---
- $T(n): n*n$ 크기의 행렬 A, B를 곱하는데 걸리는 시간
- 단위  연산: 곱셈하는 연산
- 입력 크기: 행과 열의 수, n
- *모든 경우* 시간복잡도 분석: threshold를 1이라고 할 때(theshold는 차수에 전혀 영향을 미치지 않는다).
- 재현식: $T(n)=8T(\frac{n}{2}), n>1$이고, $n=2^k(k\ge 1)$. $T(1)=1$
	- 이 식을 전개해 보면,
		- $T(n)=8*8*...*8$(k번)
				$8^k$
				$8^{lgn}$
				$n^{lg8}$
				$n^3$
				$\in \Theta(n^3)$


### 2. Strassen I
---
- 단위 연산: 곱셈하는 연산
- 입력 크기: 행과 열의 수, $n$
- *모든 경우* 시간복잡도 분석: threshold를 1이라고 할 때.
- 재현식: $T(n)=7T(\frac{n}{2}), n>1$이고, $n=2^k(k\ge 1)$. $T(1)=1$
	- 이 식을 전개해 보면,
		- $T(n)=7*7*...*7$(k번)
				$7^k$
				$7^{lgn}$
				$n^{lg7}$
				$n^{2.81}$
				$\in \Theta(n^{2.81})$


### 3. Strassen II
---
- 단위 연산: 덧셈과 뺄셈 연산
- 입력 크기: 행과 열의 수, $n$
- *모든 경우* 시간복잡도 분석: threshold를 1이라고 할 때.
- 재현식: $T(n)=7T(\frac{n}{2})+18(\frac{n}{2})^2), n>1$이고, $n=2^k(k\ge 1)$. $T(1)=0$
	- 도사정리의 3가지 중에서 1번을 이용하면 간단히 해를 구할 수 있다.
		- $T(n)=6n^{lg_27}-6n^2$
				$6^{2.81}-6n^2$
				$\in \Theta(n^{2.81})$
